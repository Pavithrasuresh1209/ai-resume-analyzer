AI Resume Analyzer

**Customized AI-based Resume Analyzer Web App**

This project allows users to upload their resumes and receive an automated analysis with a scoring system. The app identifies strengths, weaknesses, and suggestions to improve the resume. The UI has been enhanced for a clean and modern look

**Features**

1.Upload and analyze resumes in PDF format
2.Score and provide feedback for ATS (Applicant Tracking System) compatibility
3.User-friendly interface with drag-and-drop upload
4.Responsive design for desktop and mobile

**Tools & Technologie**s

Frontend: React, TypeScript, Tailwind CSS, Zustand
Backend/Utilities: pdfjs-dist, react-dropzone
Build Tools: Vite, React Router
